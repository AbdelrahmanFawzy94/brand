datepicker Forms

Use form to present your design and content as clearly and efficiently as possible.

1- import SharedDatepickerComponent in ts file

import { SharedDatepickerComponent } from '@shared';

2- import GetControlPipe in ts file

import { SharedDatepickerComponent, GetControlPipe } from '@shared';

3- add to imports modules in module or stand alone component

imports: [ SharedDatepickerComponent ];

4- add to imports modules in module or stand alone component pipe (extra step)

imports: [ SharedDatepickerComponent, GetControlPipe ];

5- in html file declare input element with pipe

<lib-datepicker [passedFormControl]="form | getControl : 'controlName'" label="test label"></lib-datepicker>

or without pipe and normal control (not inside any form group)

<lib-datepicker [passedFormControl]="formControl" label="test label"></lib-datepicker>
