Forms

Use form to present your design and content as clearly and efficiently as possible.

1- import SharedCheckboxComponent in ts file

import { SharedCheckboxComponent } from '@shared';

2- import GetControlPipe in ts file

import { SharedCheckboxComponent, GetControlPipe } from '@shared';

3- add to imports modules in module or stand alone component

imports: [ SharedCheckboxComponent ];

4- add to imports modules in module or stand alone component pipe (extra step)

imports: [ SharedCheckboxComponent, GetControlPipe ];

5- in html file declare input element with pipe

<shared-checkbox label="first checkbox" [passedFormControl]="form | getControl : 'firstControl'" (onToggle)="onToggle($event)"></shared-checkbox>

or without pipe and normal control (not inside any form group)

<shared-checkbox label="first checkbox" [passedFormControl]="formControl" (onToggle)="onToggle($event)"></shared-input>
