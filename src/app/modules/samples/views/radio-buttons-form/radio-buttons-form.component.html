Forms

Use form to present your design and content as clearly and efficiently as possible.

1- import SharedRadioButtonsComponent in ts file

import { SharedRadioButtonsComponent } from '@shared';

2- import GetControlPipe in ts file

import { SharedRadioButtonsComponent, GetControlPipe } from '@shared';

3- add to imports modules in module or stand alone component

imports: [ SharedRadioButtonsComponent ];

4- add to imports modules in module or stand alone component pipe (extra step)

imports: [ SharedRadioButtonsComponent, GetControlPipe ];

5- in html file declare input element with pipe

<shared-radio-buttons labelkey="nameEn" name="name" labelPosition="before" [passedFormControl]="form | getControl : 'firstControl'" [radioButtonsList]="radioButtonsList" (onSelection)="onSelection($event)"></shared-radio-buttons>

or without pipe and normal control (not inside any form group)

<shared-radio-buttons labelkey="nameEn" name="name" labelPosition="before" [passedFormControl]="formControl" [radioButtonsList]="radioButtonsList" (onSelection)="onSelection($event)"></shared-radio-buttons>
