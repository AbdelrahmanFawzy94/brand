Forms

Use form to present your design and content as clearly and efficiently as possible.

1- import SharedInputComponent in ts file

import { SharedInputComponent } from '@shared';

2- import GetControlPipe in ts file

import { SharedInputComponent, GetControlPipe } from '@shared';

3- add to imports modules in module or stand alone component

imports: [ SharedInputComponent ];

4- add to imports modules in module or stand alone component pipe (extra step)

imports: [ SharedInputComponent, GetControlPipe ];

5- in html file declare input element with pipe

<shared-select [passedFormControl]="form | getControl : 'controlName'" label="test label"></shared-select>

or without pipe and normal control (not inside any form group)

<shared-select [passedFormControl]="formControl" label="test label"></shared-select>
